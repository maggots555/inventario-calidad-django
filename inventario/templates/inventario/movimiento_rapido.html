Instrucciones de Uso
📱 Con Scanner de Cámara:
  1. Haz clic en "Activar Scanner de Cámara"
  2. Permite acceso a la cámara cuando se solicite
  3. Coloca el código de barras/QR dentro del marco azul
  4. El código se detectará y completará automáticamente
  5. El modal se cerrará automáticamente al detectar el código
Android/iOS: Para usar la cámara necesitas conexión HTTPS. Si no funciona, usa entrada manual.
⌨️ Entrada Manual:
  1. Escribe el código QR/barras en el campo de texto
  2. El sistema verificará que exista en el inventario
  3. Completa los datos del movimiento
  4. Haz clic en "Registrar Movimiento"
Recomendado: Funciona en todos los navegadores y dispositivos.
Scanner Universal: Detecta automáticamente códigos QR y códigos de barras (EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, Codabar, ITF). Para dispositivos móviles se requiere HTTPS para usar la cámara.
Scanner Universal - QR y Códigos de Barras
Scanner Universal: Detecta automáticamente códigos QR y códigos de barras. Coloca el código dentro del marco azul para escanearlo.
📱 Códigos QR:
• Cuadrados con puntos
• Datos de texto/URL
• Lectura 360°
📊 Códigos de Barras:
• Líneas verticales
• EAN, UPC, Code 128
• Lectura horizontal
