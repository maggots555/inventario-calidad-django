Instrucciones de Uso del Scanner
📱 Con Scanner de Cámara:
  1. Haz clic en "Scanner" junto al campo código QR
  2. Permite acceso a la cámara cuando se solicite
  3. Coloca el código de barras/QR dentro del marco azul
  4. El código se detectará y completará automáticamente
  5. Se buscará automáticamente el producto fraccionario
Android/iOS: Para usar la cámara necesitas conexión HTTPS. Si no funciona, usa entrada manual.
⌨️ Entrada Manual o Scanner Físico:
  1. Escribe o escanea el código QR/barras en el campo
  2. El sistema buscará automáticamente mientras escribes
  3. Verifica la información del producto fraccionario
  4. Completa los datos del consumo
  5. Haz clic en "Registrar Movimiento Fraccionario"
Recomendado: Funciona con scanners físicos automáticamente.
Productos Fraccionarios: Solo productos configurados como fraccionarios (líquidos, granulados, etc.) pueden ser procesados aquí. Los productos normales se redirigen automáticamente al movimiento estándar.
Scanner Universal - QR y Códigos de Barras
Scanner para Productos Fraccionarios: Detecta automáticamente códigos QR y códigos de barras. Coloca el código dentro del marco azul para escanearlo.
📱 Códigos QR:
• Cuadrados con puntos
• Datos de texto/URL
• Lectura 360°
📊 Códigos de Barras:
• Líneas verticales
• EAN, UPC, Code 128
• Lectura horizontal
