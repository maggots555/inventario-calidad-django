{% comment %} _sky_flames.html — Marco decorativo SVG de electricidad animada para skyscrapers EXPLICACIÓN PARA PRINCIPIANTES: La electricidad se crea con y la técnica stroke-dashoffset: - stroke-dasharray define un segmento corto (el "rayo") y un hueco largo - Al animar stroke-dashoffset, el rayo viaja a lo largo del borde - Una segunda animación de opacity crea el parpadeo eléctrico intermitente Cada borde tiene dos rayos desfasados medio ciclo para mayor riqueza visual. Cada rayo tiene dos capas: glow (grueso, difuso) y core (fino, brillante). Animaciones definidas en seguimiento_cliente.css (seTravelDown/Up/Right, seFlickerGlow/Core). {% endcomment %}
