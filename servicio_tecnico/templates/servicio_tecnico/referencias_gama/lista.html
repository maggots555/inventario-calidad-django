¿Cómo funciona la clasificación automática?

Cuando creas una orden de servicio, el sistema busca coincidencias en esta tabla:

  • Coincidencia exacta: Marca y modelo idénticos
  • Coincidencia parcial: Si el modelo ingresado contiene el modelo base registrado
  • Ejemplo: Si registras "Lenovo ThinkPad", coincidirá con "Lenovo ThinkPad X1 Carbon"
  • Sin coincidencia: El usuario puede seleccionar la gama manualmente
